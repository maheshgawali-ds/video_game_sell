video_game_sell
 🎮 Video Game Sales Data Analysis (1980–2024)

 📊 Project Overview
This project analyzes video game sales data to identify trends, top-performing genres, and platforms.
 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

🔍 Key Insights
- Sales increased rapidly between 2000 and 2008, then declined after 2009
- Action genre has the highest sales
- North America contributes the most to total sales

 📁 Files
- Jupyter Notebook (.ipynb)

 🚀 Conclusion
This project helped me understand real-world data analysis using Python.
